# Mybrary(Android)

----------

- 개발기간: 2018.11.06 ~ 2018.12.17
- 개발목적: 도서 관련 SNS
- 주요기능
  - 실시간 베스트셀러 순위
  - 도서 검색 & 바코드 인식 & 찜목록 기능
  - 게시글 작성 & 수정 & 삭제
  - 친구 목록 & 팔로우 & 팔로워 & 알림 기능
  - 다른 사람 게시글 조회 & 좋아요 & 댓글
  - 채팅 기능 & 챗봇 기능

----------

## 사용 도구

- Android Studio (Java)
- 알라딘 API
- Firebase
- Dialogflow
